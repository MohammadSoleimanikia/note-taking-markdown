# Note app with support of markdown (TypeScript + React)



## Features

- **Add and manage Notes**  
  Create custom Notes with markdown support and add tag for better organizing

- **search by title and tags**  
  search notes by title and tags.

- **Responsive UI**  
  A fully responsive interface optimized for both desktop and mobile devices.

- **use React-router**  
  use React-router for routing and making single page app

- **Persistent Data with Local Storage**  
  All Notes are saved and restored automatically using the browser’s Local Storage.

---

## Tech Stack

- **React + TypeScript**
- **Vite**
- **react-router-dom**
- **Local Storage**
- **Tailwind**

---

## Getting Started
```bash
# Clone the repository
git clone <repo-url>

# Navigate into the project folder
cd budget-app

# Install dependencies
npm install

# Run the development server
npm run dev
```
